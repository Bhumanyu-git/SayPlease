# SayPlease Documentation
## Rule #1
Keyword `please` mandatory in each line of code.  
For example: `please display`  
Note: Functions to be discussed in later tutorial.
## Rule #2
SayPlease is a case senisitive language. It means `please` and `Please` are not same.
## Function #1: `display`
`display` function helps you to display text in the output.  
For example: ```please display  Polite  Text:hello world  hello world```
## Concept #1: Variable
Variable is a container that can store data. You get only one variable to store data. It means storing data deletes the previous one. The variable is called `var`.  
Note: The default value of `var` is `0`.
## Funtion #2: `edit variable`
`edit variable` helps you to edit the `var` to store data.  
For example: `please edit variable  Polite  Data:hello world  var = hello world`
## Function #3: `var`
`var` functions helps you to view the data stored in the `var`.  
For example: `please var  Polite  hello world`
## Function #4: `quit`
`quit` helps you to stop the execution of the program.  
For example: ` please quit  Polite  1`  
Note: The `1` indicates the successful execution of the program and `0` indicates unsuccessful execution of the program.
## Function #5: `recite a poem`
`recite a poem` makes the language recite a poem.  
Although it looks weird when a programming language recites a poem, but, yes, SayPlease does.  
For example: `please recite a poem  Polite  You built the line,\nYou wrote it well,\nBut missed the charm,\nThat breaks the spell.`
## Concept #2: Math
SayPlease like other programming languages, does helps you to calculate.  
We have two functions to do the same: `calculate` and `valculate`. Both have similarities and differences.  
Note: These functions to be discussed in later tutorials.
## Functions #6: `calculate`
`calculate` helps you to solve math problems.  
For example: `please calculate  Polite  Expression:1+1  2`
## Function #7: `valculate`
`valculate` is same as `calculate` except that the output is stored in `var`.  
For example: `please valculate  Polite  Expression:1+1  2  var = 2`
## Concept #3: Time
There are functions like `nap`, `time` and `wait` that are to be discussed in later tutorial.
## Function #8: `nap` (Time)
`nap` helps you to pause the execution of the program for a definite time. When the time period is over, the program resumes the execution.  
For example: `please nap  Polite  Duration:2  <2 second pause>  The code woke up!`  
Note: The duration to be provided in seconds.
## Function #9: `say`
`say` makes the language speak something.  
For example: `please say  Polite  Text:hello world`
## Functions #10: `len`
It gives you the length of a text.  
For example: `please len  Polite  Text:hello  5`
## Function #11: `time` (Time)
It tells the current time.  
For example: `please time  Polite  10:22:32`
## Function #12: `clear`
It clears the output terminal.  
For example: `please clear`
